# Json Creation - Swap Tool

## Project Information
This project provides a tool for creating JSON structures and performing swap operations in various formats. It's designed to streamline the workflow of developers handling JSON data.

## Features
- **JSON Creation**: Easily create valid JSON structures.
- **Data Swapping**: Swap values within JSON objects seamlessly.
- **User-friendly Interface**: Intuitive interface for better user experience.

## Installation
To install the Json Creation - Swap Tool, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/AnkitMourya12/Json_Creation_-Swap_tool.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Json_Creation_-Swap_tool
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To use the tool after installation, run the following command:
```bash
node app.js
```
Make sure to provide the required inputs as prompted.
